Проект: SitDownPls — интернет-магазин мебели

Макет в Figma: https://www.figma.com/design/ZPj1HpZoklEXl7hKmYPGd2/sdp.ru

Описание:
Данный проект представляет собой создание интернет-магазина мебели для компании SitDownPls,
которая уже 15 лет работает на рынке. Цель проекта - обновить дизайн и функционал 
существующего сайта, повысить удобство и доступность для пользователей.

Ключевые задачи:
Переработка дизайна: Реализовать дизайн-макеты, разработанные штатным дизайнером компании, 
на основе имеющейся информации о 90% используемых UI-элементов.

Адаптивная вёрстка: Обеспечить корректное отображение сайта на различных устройствах 
(компьютер, планшет, мобильное устройство).

Кросс-браузерная совместимость: Сайт должен корректно отображаться в популярных браузерах 
(Chrome, Firefox, Safari, Opera, Edge).

Функциональность: Реализовать ключевые функциональные элементы магазина, такие как главная 
страница, каталог товаров, карточки товаров, поиск, корзина, и др.

Особенности проекта:
Использование современных технологий: Gulp, Sass/Scss, БЭМ-именование классов.
Оптимизация производительности: Минимизация кода, использование флексбокса и/или 
CSS Grid-вёрстки.

Обеспечение доступности: Реализация юзабилити для пользователя, соответствие стандартам 
доступности.

Этапы реализации:
Анализ требований: Изучение ТЗ, разработка плана работ.

Вёрстка: Поблочная вёрстка всех страниц, включая адаптивную версию.

Тестирование: Проверка работоспособности и корректного отображения сайта в различных 
браузерах и на разных устройствах.

Документирование: Создание документации с описанием использованных технологий, 
функциональных особенностей и инструкцией по использованию.

Результат проекта:
Полностью функциональный, адаптивный, кросс-браузерный сайт интернет-магазина мебели 
SitDownPls, соответствующий заданным требованиям и обеспеченный необходимой документацией.
